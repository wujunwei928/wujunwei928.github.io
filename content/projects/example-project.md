## 项目简介

这是我的个人技术博客，用于分享编程经验和学习笔记。

## 技术栈

| 组件 | 技术 |
|------|------|
| 静态生成器 | Hugo Extended |
| 主题 | PaperMod |
| 部署 | GitHub Pages |
| CI/CD | GitHub Actions |

## 功能特性

- 响应式设计，支持移动端
- 深色/浅色模式自动切换
- 全站搜索功能
- 代码语法高亮
- RSS 订阅支持
